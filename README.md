UI Test Automation – Phonebook Web Project
🔧 Tech Stack
Java
Selenium WebDriver
TestNG
Gradle
📖 Project Description

This project contains automated UI tests for a Phonebook web application.
The tests are designed to validate core functionalities such as adding, searching, and updating contacts.

The project follows the Page Object Model (POM) design pattern to ensure code readability, maintainability, and scalability.

✅ Test Coverage
User registration and login
Add new contact
Search contact
Update contact details
Basic validation scenarios (positive & negative)
▶️ How to Run Tests
Clone the repository
Open the project in your IDE
Run the following command:
gradle test
📊 Purpose

The goal of this project is to demonstrate practical skills in UI test automation, including test design, framework structure, and working with modern automation tools.
