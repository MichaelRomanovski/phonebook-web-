UI Test Automation – Phonebook Web Project

##  Tech Stack
- Java  
- Selenium WebDriver  
- TestNG  
- Gradle  

---

##  Project Description

This project contains automated UI tests for a Phonebook web application.  
It covers core user workflows such as contact management and authentication.

The framework is built using the Page Object Model (POM) design pattern, ensuring scalability, maintainability, and clean test structure.

---

 Test Coverage:

- User registration and login  
- Add new contact  
- Search contact  
- Update contact details  
- Validation scenarios (positive & negative cases)  

---

##  Framework Features

- Page Object Model (POM) architecture  
- Reusable page components  
- Clear test structure with TestNG  
- Easy test execution with Gradle  
- Separation of test logic and UI elements  

---

##  How to Run Tests

1. Clone the repository  
2. Open the project in your IDE
